# Portfólio

Bem-vindo ao repositório do meu portfólio!

## Descrição

Este portfólio foi desenvolvido com base no modelo fornecido pela Alura, como parte do programa de formação front-end da turma 6 da ONE. Utilizei o layout disponibilizado no Figma como referência e fiz algumas modificações para personalizá-lo.

## Funcionamento

Você pode conferir o portfólio clicando [aqui](https://vitorvalentimsilva.github.io/Portfolio/)

- **HTML/CSS/JavaScript:** Utilizei essas tecnologias fundamentais para o desenvolvimento web.
- **Figma:** O layout do portfólio foi baseado no modelo fornecido pelo Figma.
